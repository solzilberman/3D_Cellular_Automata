## 3D Cellular Automata in C++ with OpenGl

 **Compile**
> `make` <br />

**Run**

> `make run` <br />
> or <br />
> .`/main [animation enabled] [rotation enabled]` <br />
> example usage: <br />
>  `./main true false` <br />

**Dependencies**
> GLSL >= 3.30 <br />
> glm [library](http://glm.g-truc.net/0.9.6/index.html) <br />
